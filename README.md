# 📊 Global Sales Performance Dashboard (Power BI)

## 📌 Overview
This project showcases a Global Sales Performance Dashboard built using Microsoft Power BI.  
The dashboard analyzes 4 years of global sales data (2016–2019) and provides interactive visualizations to explore sales performance, trends, and distribution.

## 🎯 Objectives
- Transform raw sales data into meaningful insights  
- Build an interactive and user-friendly dashboard  
- Analyze sales performance across time, categories, and regions  


## 🛠️ Tools & Technologies
- Microsoft Power BI  
- Power Query (Data Cleaning & Transformation)  
- DAX (Data Analysis Expressions)  


## 🔧 Project Workflow
1. Data Collection (Course dataset)  
2. Data Cleaning & Preprocessing  
3. Data Transformation using Power Query  
4. Data Modeling & Relationships  
5. DAX Measures for KPIs and calculations  
6. Dashboard Design & Visualization  


## 📊 Dashboard Features
- KPI Cards: Sales, Profit, Quantity  
- Growth Indicator with percentage change  
- Monthly Sales Trend (Line Chart)  
- Product Revenue (Top Countries)  
- Weekend vs Weekday Sales Analysis  
- Category-wise Revenue Distribution  
- Top Products Analysis  
- Interactive Year Filter (2016–2019)  

## 🚀 Key Learnings
- Data cleaning and transformation techniques  
- Writing DAX measures for business KPIs  
- Designing interactive dashboards  
- Data storytelling using visuals
